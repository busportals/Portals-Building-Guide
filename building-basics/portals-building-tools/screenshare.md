# Screenshare

### Overview

The Screen Share tool allows room admins to broadcast a live screen, window, or Chrome tab directly into the space. It’s useful for presentations, livestreams, collaborative work, or watch parties.

* Only Chrome Tab sharing supports audio playback
* Resize the screen share object while holding it using Q (shrink) and E (expand)

### Placement Settings

#### To activate screen sharing:

* Ensure building mode is off
* As a room admin, hover over the Screen Share object
* Click the blue icon labeled “Toggle Screen Share”
* This opens the browser-native sharing prompt where you can choose to share a Chrome tab, application window, or entire screen

#### Chrome Tab Sharing (Recommended)

* Select a specific tab from the list
* Optionally toggle “Also share tab audio” to broadcast sound
* Press Share to start the broadcast

### Common Use Cases

* Hosting in-world presentations or tutorials
* Streaming videos or music for group experiences
* Sharing live browser-based applications or dashboards
