# Chart

### Overview

The Chart tool creates a 3D candlestick chart for tokens on Solana.

### Inventory Settings

* Paste Token Address – Choose the token CA for the chart you wish to display.

### Chart Settings Panel

* Token Address – Choose the token CA for the chart you wish to display
* Height - Determines the height of the candles while maintaining the current scale
* Rotation - Dictates the angle between each candle. Ideal for adding a curve to your chart.
* Collision – Enables or disables physical interaction with the surface
* Time Interval - Displays the interval each candle represents
  * Currently supports 1 minute, 15 minutes, and 1 hour time intervals

### Common Use Cases

* Displaying live token price data
* Create gamified trading experiences
