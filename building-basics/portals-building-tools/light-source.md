# Light Source

### Overview

The Light Source tool adds dynamic lighting to your space. You can customize the color, brightness, and range, and optionally set it to appear only at night.

### Placement Settings

#### Light Settings Panel

* Enable This – Turns the light on or off
* Choose Custom – Opens the color picker to set the light color
* Brightness – Controls how intense the light appears (higher = brighter)
* Range – Sets how far the light reaches from its origin
* Only Night (Toggle) – When enabled, the light only appears during night mode

### Common Use Cases

* Illuminate dark areas or highlight key features
* Create colored ambient lighting for mood or tone
* Simulate day/night lighting dynamics in your scene
