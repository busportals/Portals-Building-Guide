# NPC

### Overview

The NPC (non-playable character) tool allows you to spawn interactive characters that can use dialogue trees, AI prompts, and follow set animations or movement paths. NPCs must use rigged GLB avatars to support animation.

### Inventory Settings

NPC Upload Interface

* Paste Item URL – Link to a hosted rigged GLB avatar
* Choose from Options – Select a preset NPC model from available avatars
* Spawn NPC – Adds the NPC to your space

### Placement Settings

#### NPC Settings Panel

* NPC Name – Sets the display name of the NPC
* Auto Popup – When enabled, dialogue or interaction appears automatically when a player approaches
* Default Animation – Sets the name of the idle or looping animation (must match animation in the GLB file)
* AI Settings – Opens the AI prompt editor or dialogue tree builder
* Custom Title – Optional field for internal reference or organization

### Common Use Cases

* Dialogue-driven quests and tutorials
* Guided tours or story elements
* Background characters with idle animations
