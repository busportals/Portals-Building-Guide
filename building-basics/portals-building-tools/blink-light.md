# Blink Light

### Overview

The Blink Light tool is a dynamic light source that turns on and off at fixed intervals. It’s useful for signaling, atmosphere, or sci-fi effects like alarms or warning lights.

### Placement Settings

#### Blink Light Settings Panel

* Choose Custom – Opens the color picker to set the light color
* Brightness – Controls the intensity of the light
* Range – Sets how far the light spreads
* Only Night – If enabled, the light only appears during night mode
* Blink Interval – Time in seconds between blinks (the off duration)
* Blink Duration – Time in seconds the light stays on during each blink

### Common Use Cases

* Warning or alert systems
* Decorative blinking lights for sci-fi or party environments
* Signaling mechanisms for puzzles or timed events
