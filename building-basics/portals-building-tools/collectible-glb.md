# Collectible GLB

### Overview

The Collectible GLB tool allows you to spawn interactive 3D objects that players can collect to update values like score, inventory, or health. Ideal for pickups, coins, and reward-based gameplay.

### Inventory Settings

Collectible Upload Interface

* Paste Item URL – Direct link to a .glb file
* Enable Draco Compression – Toggle to enable Draco-compression
* Drag & Drop – Upload a .glb file from your device (max size: 15MB)
* Spawn GLB – Places the collectible in your space

### Placement Settings

#### Collectible Settings Panel

* Shadow On – Toggle shadow rendering for the object
* Size – Adjusts the scale of the GLB
* GLB URL – Displays the hosted file URL
* Animation On – Toggles built in collectible animation
* Sound Effect URL – Add a sound (via URL or upload) to play on collection
* Local Animation – Plays animation only on the player’s client
* Value to Update – Specify which value this collectible modifies (e.g., health, score)
* Operation – Choose to Add To, Set, or Subtract From the value
* Value Change – Amount to modify the value by
* Auto Display Value – Toggles showing the updated value on collection
* Min/Max Respawn Time – Time range for the collectible to reappear
* Min/Max Despawn Time – Time range for the collectible to disappear
* Random Spawn Radius – Controls how far from the original spot it can randomly respawn
* Custom Title – Optional internal label

#### Common Use Cases

* Reward systems for minigames
* Health, currency, or score pickups
* Dynamic collectibles for exploration or quests
