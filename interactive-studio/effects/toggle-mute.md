# Toggle Mute

## Use Cases

***

* Turn on a players mic.

## Configuration

***

| Setting     | Permitted Values | Description                                                                                            |
| ----------- | ---------------- | ------------------------------------------------------------------------------------------------------ |
| Mute Player | Toggle           | When toggled on this effect will mute the player, when toggled off this effect will unmute the player. |
