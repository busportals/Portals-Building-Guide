# Record NPC Path

## Use Cases

***

* Create cut scenes with the NPCs.

## Configuration

***

| Setting            | Permitted Values      | Description                                                                                                                        |
| ------------------ | --------------------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| Record Player Path | Player recorded path. | Press the record button to begin recording your path. The NPC will copy your path. When you are done press X to confirm your path. |
| Preview Transition | n/a                   | Check to see if the path looks correct.                                                                                            |
| Loop               | toggle                | Loops the path.                                                                                                                    |
