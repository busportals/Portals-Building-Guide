## Use Cases

***

* Sending a tag event to an NPC
* Getting an NPC to initiate a conversation

## Configuration

***

| Setting                     | Permitted Values | Description                                                                                                          |
| --------------------------- | ---------------- | -------------------------------------------------------------------------------------------------------------------- |
| NPC Name                    | String           | Type the exact name of the NPC you want to send a message to.                                                        |
| Message                     | String           | Type in the message to send. You can use tag events in the message as well.                                          |
| Player must be within range | Toggle           | When this is toggled on the player will be required to be within close proximity of the NPC for the message to send. |
