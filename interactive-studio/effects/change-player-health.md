# Change Player Health

## Use Cases

***

* Kill players
* Heal players
* Set baseline health

## Configuration

***

| Setting       | Permitted Values                                                             | Description                                       |
| ------------- | ---------------------------------------------------------------------------- | ------------------------------------------------- |
| Operation     | <p>Drop Down:<br>- Set Player Health<br>- Heal Player<br>- Damage Player</p> | Choose how you want to effect the players health. |
| Health Change | Integer                                                                      | Enter the value to use in the operation.          |
