# Play Sound In A Loop

## Use Cases

***

* Great for background noises (fires, rain, crickets, etc.)

## Configuration

***

| Setting               | Permitted Values    | Description                                                                                                                                  |
| --------------------- | ------------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| URL                   | URL to an MP3 file. | Paste in a link of a hosted URL file.                                                                                                        |
| Distance              | Numerical values    | A negative value will make the sound play globally. Any positive number will be proximity audio based on the origin of the object.           |
| Preload               | toggle              | When this is on it will load the audio file before it needs to play. (on room load)                                                          |
| Play on Start         | toggle              | When this is on the audio can be played when a user enters the space.                                                                        |
| Volume                | slider              | Set the volume of the audio.                                                                                                                 |
| Random Pitch Variance | slider              | When this is used the audio file's pitch will be randomly adjusted based on the variance set. A higher value will lead to a higher variance. |
| Preview Sound         | n/a                 | Click the preview button to test you sounds volume and variance.                                                                             |
