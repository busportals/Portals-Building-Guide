# Change Bloom

## Use Cases

***

* Change Fog

## Configuration

***

| Setting         | Permitted Values | Description                                                                               |
| --------------- | ---------------- | ----------------------------------------------------------------------------------------- |
| Bloom Intensity | Slider           | The strength of the bloom effect, determining how bright and pronounced the glow appears. |
| Bloom Diffusion | Slider           | The spread or softness of the bloom effect, influencing how widely the glow disperses.    |
| Bloom Clamp     | Slider           | The maximum brightness level allowed for the bloom effect to prevent excessive glare.     |
