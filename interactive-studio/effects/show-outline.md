# Show Outline

## Use Cases

***

* Highlighting an object
* Helping signal to users which object they need

## Configuration

***

| Setting       | Permitted Values | Description                                         |
| ------------- | ---------------- | --------------------------------------------------- |
| Width         | Slider           | Sets the px width of the outline shown              |
| Outline Color | Color Selector   | Select the color of the outline you wish to display |
