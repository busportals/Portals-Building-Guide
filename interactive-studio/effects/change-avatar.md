# Change Avatar

## Use Cases

***

* Great for creating themed experiences (e.g. Gecko Space Run with gecko avatar)

## Configuration

***

| Setting    | Permitted Values | Description                                                              |
| ---------- | ---------------- | ------------------------------------------------------------------------ |
| Avatar URL | String           | Enter a glb avatar URL.                                                  |
| Persistant | toggle           | Toggling this on will make the avatar persist if the user changes rooms. |
