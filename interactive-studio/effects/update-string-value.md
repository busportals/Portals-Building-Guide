# Update String Value

## Use Cases

***

* Storing a player's name or custom display text.
* Setting status messages (e.g. "Ready", "Waiting", "Complete").
* Updating labels or custom text that appears in your space.
* Storing text data for use in other triggers or effects.

## Configuration

***

| Setting      | Permitted Values | Description                                                        |
| ------------ | ---------------- | ------------------------------------------------------------------ |
| Value Label  | String           | The name of your string variable. (e.g. playerName, statusText, etc.) |
| Target Text  | String           | The text value to set the variable to.                             |
