## Use Cases

***

* Creating cut scenes
* Forcing users to look at an object (like a hint)

## Configuration

***

| Setting                 | Permitted Values          | Description                                                                                                                                                                                                       |
| ----------------------- | ------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Unlock After Time       | toggle                    | When this is toggled on the camera will automatically unlock after the specified amount of time. If this is turned off the camera will remain locked, you will need to use an unlock effect to unlock the camera. |
| Seconds Before Unlock   | slider or numerical input | Time in seconds before the camera unlocks.                                                                                                                                                                        |
| Allow Effector on start | toggle                    | If this is on, when a user enters a room their camera can instantly be locked to an object.                                                                                                                       |
