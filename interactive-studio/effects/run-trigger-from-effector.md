# Run Trigger From Effector

## Use Cases

***

* Great for reseting tasks
* Creating checkpoints
* Randomized outcomes

## Configuration

***

| Setting      | Permitted Values | Description                                                                                    |
| ------------ | ---------------- | ---------------------------------------------------------------------------------------------- |
| Task Name    | Drop down list.  | Enter the task name that you would like to change the state of.                                |
| Is Set To... | Drop down list.  | Set the state change you would like to take place. This behaves like a normal trigger setting. |
| Delay        | Numerical Values | Set the delay of when the trigger takes place.                                                 |
