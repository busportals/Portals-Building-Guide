# Move Item

## Use Cases

***

* Opening a gate
* Moving objects in obstacle course

## Configuration

***

| Setting                          | Permitted Values                                                    | Description                                                                                                                                   |
| -------------------------------- | ------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Destination                      | Position, Rotation, Size numerical inputs or set using build tools. | You can set the exact Position, Rotation, or Size an object moves/transforms to. You can set by typing in values or setting with build tools. |
| Transition Duration              | Slider or Integer input                                             | The time it takes for the object to move from its current position to the end position.                                                       |
| Preview Transition               | n/a                                                                 | Click the Preview Transition button after setup to see how your movement looks.                                                               |
| Reset Destination                | n/a                                                                 | Set the object to its original state.                                                                                                         |
| Optimize For Collision Detection | toggle                                                              | If the object is going to have "On Collision" triggers it is recommended to toggle this on. It will increase load on the space.               |
