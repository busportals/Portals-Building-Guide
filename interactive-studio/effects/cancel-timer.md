# ⏲️ Cancel Timer

## Use Cases

***

* Invalidate a timer when a user goes out of bounds
* Cancel a timer when a user exits a game

## Configuration

***

| Setting    | Permitted Values | Description                                                                                               |
| ---------- | ---------------- | --------------------------------------------------------------------------------------------------------- |
| Timer Name | String           | Enter the name of the timer you wish to cancel. A corresponding timer must be started and running to use. |
