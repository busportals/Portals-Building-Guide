# Change Movement Profile

## Use Cases

***

* Changing player speed when they get "stuck in honey" or hit a "speed boost"

## Configuration

***

| Setting                | Permitted Values | Description                                        |
| ---------------------- | ---------------- | -------------------------------------------------- |
| Movement Profile Label | String           | Enter the name of a movement profile to switch to. |
