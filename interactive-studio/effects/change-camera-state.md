## Use Cases

## Configuration

***

| Setting           | Permitted Values          | Description                                                                                                     |
| ----------------- | ------------------------- | --------------------------------------------------------------------------------------------------------------- |
| Camera State Name | string                    | Enter the name of a camera state. You can setup cameras by going to: space options > camera > add camera state. |
| Smooth Transition | toggle                    | Toggling this on will create a transition between camera states. If it is off the transition will be instant.   |
| Transition Speed  | slider or numerical input | Set the speed of the transition between camera states.                                                          |
