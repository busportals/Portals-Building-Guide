# Teleport

## Use Cases

***

* Respawning a user as part of their "death" in a game
* Moving players to new locations in a space

## Configuration

***

| Setting    | Permitted Values | Description                                                                                   |
| ---------- | ---------------- | --------------------------------------------------------------------------------------------- |
| Room ID    | String           | This will be the Room ID located in the spaces URL.                                           |
| Spawn Name | String           | Type in the spawn name. You will need to have a spawn point setup and named for this to work. |
