# Display Value

## Use Cases

***

* Displaying scores in games.
* Displaying states (e.g. lives, bullets, potions)

## Configuration

***

| Setting     | Permitted Values | Description                                               |
| ----------- | ---------------- | --------------------------------------------------------- |
| Value Label | String           | The label of your value. (e.g. score, lives, coins, etc.) |
| Value Color | Color Selector   | The text color of your displayed value.                   |
