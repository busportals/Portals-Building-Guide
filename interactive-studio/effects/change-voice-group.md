# Change Voice Group

## Use Cases

***

* Create meeting rooms
* Create team voice channels

## Configuration

***

| Setting    | Permitted Values | Description                                                                                                                           |
| ---------- | ---------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| Group Name | text string      | Enter any text stream to create the "name" of the voice group. Living the value blank will change it back to the default voice group. |
