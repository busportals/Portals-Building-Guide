# Timer Stopped

## Configuration

***

| Setting                              | Permitted Values    | Description                                                         |
| ------------------------------------ | ------------------- | ------------------------------------------------------------------- |
| When Timer Finished With The Name... | String              | Enter the name of the timer.                                        |
| And the Resulting Time Is...         | Drop Down Selection | Choose whether the user needs to be faster or slower than the time. |
| Than the time (in seconds)...        | Slider              | Enter time required                                                 |
