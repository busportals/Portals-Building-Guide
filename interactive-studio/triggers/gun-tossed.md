# Gun Tossed

## Use Cases

***

* Reset player state when disarmed
* Hide weapon-related UI elements
* End a game mode that requires holding a weapon
* Track weapon drop statistics

## Configuration

***

No additional configuration beyond the base configuration parameters. This trigger fires when a player drops or unequips the gun, typically via the Toss Gun effect.
