# Item Collected

## Configuration

***

| Setting                       | Permitted Values    | Description                                               |
| ----------------------------- | ------------------- | --------------------------------------------------------- |
| With An Item Name Of...       | String              | Input the exact item name.                                |
| And This Many Is Collected... | Integer             | Choose the quantity to be compared by the operation type. |
| Operation Type                | Drop Down Selection | Select the operator logic.                                |
