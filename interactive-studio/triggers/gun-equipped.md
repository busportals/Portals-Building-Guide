# Gun Equipped

## Use Cases

***

* Start a game mode when player picks up a weapon
* Display UI elements or instructions when armed
* Play equip sound or animation
* Track weapon pickup statistics

## Configuration

***

No additional configuration beyond the base configuration parameters. This trigger fires when any player equips the gun it is attached to.
