# Gun Shot Hit

## Use Cases

***

* Track hit accuracy statistics
* Play hit confirmation sounds
* Award points for successful hits
* Trigger visual effects on hit

## Configuration

***

No additional configuration beyond the base configuration parameters. This is a global trigger that fires whenever a shot from this gun type hits any damageable target (players, Combat NPCs, or destructibles).
