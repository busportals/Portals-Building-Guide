# Gun Stopped Aiming

## Use Cases

***

* Reset camera zoom or FOV
* Hide scope overlay UI
* Restore normal player movement speed
* Play un-aim sound effect

## Configuration

***

No additional configuration beyond the base configuration parameters. This trigger fires when a player releases the aim input (right-click on mouse, left trigger on controller).
