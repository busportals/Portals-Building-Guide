# Gun Started Aiming

## Use Cases

***

* Zoom camera or change FOV when aiming
* Display scope overlay UI
* Slow player movement while aiming
* Play aiming sound effect

## Configuration

***

No additional configuration beyond the base configuration parameters. This trigger fires when a player begins aiming (right-click on mouse, left trigger on controller).
