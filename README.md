# Portals Building Guide

[![GitHub stars](https://img.shields.io/github/stars/busportals/Portals-Building-Guide?style=flat-square)](https://github.com/busportals/Portals-Building-Guide/stargazers)
[![GitHub forks](https://img.shields.io/github/forks/busportals/Portals-Building-Guide?style=flat-square)](https://github.com/busportals/Portals-Building-Guide/network/members)
[![Built with GitBook](https://img.shields.io/badge/Built%20with-GitBook-2A2A2A?style=flat-square)](https://www.gitbook.com/)

Portals is a no-code platform for building browser-based games and interactive spaces. This guide collects the official building docs, from first-time setup to advanced logic and effects.

Use this if you're:
- new to Portals and need the basics
- an experienced builder looking for advanced tooling and patterns
- a competition participant or creator assembling more complex spaces
